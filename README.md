# User List App

This project is a simple React application that displays a list of users by consuming the JSONPlaceholder API.

## Features

- Fetches a list of users from JSONPlaceholder API using Axios.
- Displays user information, including name, email, and username.
- Uses React hooks (`useState` and `useEffect`) for state management and API calls.
- Styled with basic CSS.
